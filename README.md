MemoryMatchCardGame

    A simple memory matching game built with React.js. Flip cards, find matching pairs, and clear the board!

Live Site : https://dharanisiva78.github.io/MemoryMatchCardGame/

Features

    Randomized card shuffling

    Smooth flip animations

    Match detection

    Simple, clean UI

    No external libraries

Install dependencies:

    npm install 

Running the Project :

    npm start
